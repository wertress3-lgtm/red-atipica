# RED ATÍPICA - versión final corregida

Esta versión incorpora las correcciones indicadas en el PDF final:

- Logo oficial visible en splash, cabecera, inicio y match.
- Talentos destacados actualizados.
- Fotografías de perfil separadas correctamente de las fotografías de portafolio.
- Portafolios asociados a cada talento.
- Eliminación de rótulos sobre las imágenes de portafolio.
- Proyectos con sus logos e imágenes correctas, incluyendo DOSLOBOS.
- Calificaciones solo con estrellas y promedio, sin comentarios.
- Servicios RED ATÍPICA con precios, descripciones e imágenes.
- Sistema de sanciones y seguridad detallado.
- Filtro por región conservado.
- Preparado para GitHub y Render.

## Render

Build command:

```text
pip install -r requirements.txt
```

Start command:

```text
gunicorn app:app
```
