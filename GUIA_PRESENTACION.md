# Guía rápida de presentación — RED ATÍPICA V6

1. Iniciar en pantalla de bienvenida.
2. Seleccionar rol: profesional creativo o marca/proyecto.
3. Mostrar Inicio: proyectos destacados, talentos destacados, categorías y comunidad.
4. Abrir Explorar y demostrar el filtro por región.
5. Entrar a un proyecto: mostrar ubicación, fecha, duración, requisitos, beneficios y pago.
6. Postular y luego entrar a Postulaciones para demostrar que se guarda.
7. Mostrar Match y Mensajería.
8. Mostrar Perfil profesional con portafolio y calificaciones.
9. Mostrar Servicios RED ATÍPICA y precios.
10. Mostrar Planes para profesionales y marcas/productoras.
11. Mostrar Seguridad y sistema de sanciones.
12. Cerrar con QR: local ahora, URL alojada cuando se publique.
